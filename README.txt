Hello and thank you for your time in reviewing my code. The following scripts have been written using Python 3.8.3. 


fibonacci.py
  - From the terminal, enter the following command:
    python fibonacci.py --n "YOUR NUMBER HERE"


list.py
  - From the terminal, enter the following command:
    python list.py --original "YOUR LIST HERE" --add "YOUR LIST HERE" --delete "YOUR LIST HERE"

    NOTE: Be sure to enter your lists using double quotes around the list and single quotes around the items in the list. 
          For example, --original "['one', 'two', 'three']" --add "['four', 'five']" --delete "['two', 'five']"
